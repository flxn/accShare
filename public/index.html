FAQ


What is AccShare?

AccShare is a platform for sharing login credentials.

Why should i use it?

Everyone has probably encountered this situation:
You browse the web and stumble upon a site you want to test or that has content you want to access that is free but you have to create an account first to use it. This means you have to give out your email and name even if you are not sure wether you will use the site for more than 5 minutes.
With AccShare you can avoid this unnecessary hurdle and just log in with a shared account.
Less hassle. More privacy.

Is it free?

Yes, absolutely. But you can donate if you want to support me.


How does it work?

AccShare is completely community-driven. Users create Fake-Accounts for sites and add them to our database to help others. The more people participate the more sites can be added.

Do I always have to visit this page to use it?

Nope. There is a browser extension for Google Chrome that shows you existing logins for your current tab.
Check it out on the Chrome Web Store.

But then you can track every page I'm visiting!

Again No. There is no ID that can identify a single user and only the domain name of the current url is sent to our server. We also do not store any IPs so for AccShare you are completely anonymous (We use Google Analytics though but only for the website...).
Example:
When you visit "http://example.com/a-random-embarassing-page.html" we only see that someone requested credentials for "example.com".
